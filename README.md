# 📌 Note Card Stack App

An interactive **Note / Call Management UI** built using **HTML, CSS, and Vanilla JavaScript**.  
The application allows users to create call notes, store them in **localStorage**, and view them in a **stacked card interface** with navigation controls.

---

## 🚀 Live Demo
*https://note-card-stack-app.vercel.app/*

---

## ✨ Features

- ➕ Create new call notes using a form modal
- 💾 Persistent storage using **localStorage**
- 🃏 Card stack UI (shows only top 3 cards)
- ⬆️⬇️ Navigate cards using Up / Down buttons
- 📱 Responsive and clean interface
- 🧾 Stores:
  - Profile Image
  - Full Name
  - Home Town
  - Purpose of Call
  - Priority Category

---

## 🧠 How It Works

1. User clicks **+ button** to open the form.
2. Form data is validated.
3. The note is stored in **localStorage**.
4. Cards are dynamically generated using **DOM manipulation**.
5. The stack UI displays the **top 3 cards only**.
6. Users can rotate the stack using **Up / Down buttons**.
